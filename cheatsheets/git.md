\# Git Cheat Sheet



You can use the, git add "NameOfFIle", or git add . to add to the staging area.



Use the git commit, -m "NameOfFile" then you can add a note with, "Comment".



Extra: 

git push origin main , push to github.


git --version, check your git version.



git status, to check if all your files are saved to history or need to be added.



Use "# " to make a header.


git config --global core.editor "notepad", used to add notepad as the main editor can also use vs code with "code --wait"
